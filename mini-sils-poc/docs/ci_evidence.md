# CI Evidence

## Purpose

This document summarizes the CI(Continuous Integration) evidence for the mini SILS(Software-in-the-Loop Simulation) PoC(Proof of Concept).

The purpose of CI(Continuous Integration) in this repository is to confirm that the mini SILS(Software-in-the-Loop Simulation) can be reproduced outside the local development PC.

## CI Platform

The CI(Continuous Integration) platform is GitHub Actions.

Workflow file:

- `.github/workflows/python-test.yml`

## CI Trigger

The workflow runs on:

- push to `main`
- pull request to `main`

## CI Checks

The workflow performs the following checks:

1. Checkout repository
2. Set up Python
3. Install dependencies
4. Run pytest
5. Run mini SILS(Software-in-the-Loop Simulation)
6. Upload SILS result CSV(Comma-Separated Values) as an artifact

## Expected Test Result

The expected pytest result is:

- 13 tests passed

The tests include:

- fan control unit tests
- fan control scenario tests

## Expected SILS Result

The expected mini SILS(Software-in-the-Loop Simulation) result is:

- total: 18
- PASS: 18
- FAIL: 0

## Artifact

The CI(Continuous Integration) workflow uploads the generated result CSV(Comma-Separated Values) files as an artifact.

Artifact name:

- `fan-control-results`

The artifact contains result CSV(Comma-Separated Values) files generated by GitHub Actions, not by the local PC.

## SQA Viewpoint

From an SQA(Software Quality Assurance) viewpoint, this CI(Continuous Integration) setup provides evidence that:

- the test environment can be reconstructed automatically
- the unit tests can be executed automatically
- the SILS(Software-in-the-Loop Simulation) scenario execution can be reproduced
- PASS / FAIL results can be reviewed as generated artifacts
- local-only assumptions are reduced

## Current Baseline

The current baseline confirms:

- Python fan control logic is implemented
- pytest passes
- scenario CSV(Comma-Separated Values) execution passes
- result CSV(Comma-Separated Values) artifacts are generated
- GitHub Actions CI(Continuous Integration) succeeds
