# Comparison Evidence Policy

## Purpose

This document defines how SILS/HILS comparison evidence is managed in the Mini SILS/HILS PoC.

The objective is to ensure that comparison activities are reproducible, reviewable, and traceable.

## Required Evidence

The following comparison evidence shall exist:

* comparison_summary.txt

Additional evidence files may be added in future phases.

## Governance Rule

Comparison evidence must exist before governance checks can pass.

Missing comparison evidence shall be treated as a governance failure.

## Future Expansion

Future comparison governance may verify:

* file existence
* file format
* scenario coverage
* SILS/HILS execution consistency
* automated comparison result validation

This phase verifies existence only.
